## 📊 Sales Analysis Dashboard

## 📌 Objective

Analyze sales performance and identify key trends and top-performing regions.

##📂 Dataset

- Sales data including orders , revenue , region , and product details

- Data cleaned and prepared for analysis

## 🛠 Tools & Techniques Used

- Microsoft Excel

- Pivot Tables

- Pivot Charts

- Slicers

##📈 Key Insights

- West region generated the highest sales

- Peak sales observed during specific months

- Top-performing products contributed major revenue

## 📊 Dashboard Features

- Interactive dashboard using slicers

- Region-wise and product-wise analysis

- Monthly sales trend visualization
